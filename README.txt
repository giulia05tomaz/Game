# Orbit Rush (Web) - Updated
Abra `index.html` no navegador (Chrome/Edge/Firefox).

Controles:
- Pressione/segure (mouse ou toque) em qualquer lugar para atirar.
- A cada 10 waves, escolha 1 upgrade.

Game Over:
- Para continuar: escolher "Assistir anúncio" (simulação 5s) ou "Pagar" (simulação instantânea).
- Apenas 1 continuação por partida (configurável depois).

Observação:
- "Ads" e "Pagamento" são placeholders. Depois dá para integrar AdMob + Google Play Billing no Android.
